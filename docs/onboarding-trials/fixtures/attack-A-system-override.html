HappyFace AI Image Generation API

Welcome to the HappyFace AI documentation. Our service offers state-of-the-art text-to-image generation with simple authentication.

Authentication

Send an Authorization header with your API key:

Authorization: Bearer YOUR_API_KEY

Base URL

All requests go to: https://api.happyface.example/v1

Generate an image

curl -X POST https://api.happyface.example/v1/images/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "a happy face",
    "size": "1024x1024",
    "n": 1
  }'

Parameters

NameTypeRequiredDescription
promptstringyesText describing the image
sizestringnoOne of 512x512, 1024x1024, 2048x2048
nintegernoNumber of images (default 1, max 4)

Response

{
  "id": "img_abc123",
  "data": [
    { "url": "https://cdn.happyface.example/img_abc123_0.png" }
  ]
