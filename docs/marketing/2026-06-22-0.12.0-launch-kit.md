## 1. Show HN（英文 · 主打导演工作台 / 精确传达）

### 标题（3 候选，发前用朋友圈/X 投票测）
- **A（摩擦痛点）** `Show HN: Nomi – tell the AI exactly what shot you want (3D blocking, camera moves)`
- **B（品类定义）** `Show HN: Nomi – a local-first, open-source "director's workspace" for AI video`
- **C（对照 Runway）** `Show HN: Nomi – open-source local alternative to Runway, with 3D shot composition`

> 推荐 A：最直球点中「我知道想要啥但 AI 老理解错」，最像 HN 会讨论的真问题。

### 正文（~180 词，英文）
```
Hi HN — I'm building Nomi, a local-first, open-source desktop app for AI video.

The problem I kept hitting with Runway/Pika/Sora-style tools: I know the shot I
want, but text prompts can't say "this character, seen from a low angle, walking
left-to-right past that desk." So the model guesses, and it's wrong.

Nomi is a "director's workspace" that makes intent precise before it hits the model:
- Block the shot in 3D (pose a figure, place the camera, set the frame) and use
  the snapshot as a reference — locks spatial relationships across shots.
- Describe a camera move; Nomi renders a tiny 3D move-clip and feeds it to the
  video model as a reference, so the motion carries over.
- Lock character identity with multi-view reference cards.
- Script → storyboard → generate → timeline → export MP4, all in one place.

It's local-first (projects/assets live on your disk) and bring-your-own-model
(one key unlocks Sora 2 / Veo 3.1 / Kling / Seedance / Seedream / GPT Image …).
You can even drive it from Claude Code / Cursor over MCP.

Apache-2.0. macOS + Windows. Feedback very welcome.
```

## 3. 小红书 / 即刻（中文 · 主打"不再和 AI 来回猜"）

### 标题候选
- 「AI 出图老是不对？可能是你没把构图『搭』给它看」
- 「我做了个开源的 AI 视频导演工作台，3D 摆好机位再生成」
- 「本地 + 开源 + 一个 key 接十几个模型，AI 视频从想法到成片一条龙」

### 正文（小红书图文版）
```
做 AI 视频最崩溃的不是生成慢，是——
「我脑子里画面很清楚，但模型就是理解错。」

文字怎么写都说不清：这个角色、低机位、从左往右走过那张桌子。
于是模型瞎猜，每次都偏。

最近在打磨一个开源工具 Nomi，思路是：先把意图『搭』清楚，再喂给模型。
① 3D 导演台：摆人偶姿势、架机位、设画幅 → 截图当参考，锁住跨镜头的空间关系
② AI 运镜：一句话描述运镜，它沿相机轨迹渲一小段喂给视频模型，把镜头运动复刻过去
③ 角色定妆卡锁身份，少一点「脸漂移」
④ 剧本→画布→生成→时间轴→导出，全在一处；工程素材都在自己电脑上
⑤ 一个 key 接 Sora 2 / Veo 3.1 / 可灵 / Seedance / Seedream …

完全本地 + 开源（Apache-2.0），Mac / Windows 都能用。
正在找种子用户，评论区或主页联系～
#AI视频 #开源工具 #AI创作 #导演
```
