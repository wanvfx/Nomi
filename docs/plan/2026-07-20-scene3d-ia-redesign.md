## 2. 任务频谱（核心循环 → 长尾）

1. **每次必走**：摆场景(添加) → 选对象 → 摆相机(预览卡调画幅焦段) → 整运镜(预设为主) → 出片
2. **高频微调**：拖对象/转对象（移动/旋转）、相机位置/目标
3. **中频**：播放预览、拖播放头、走位/录 take、截图、换姿势
4. **低频**：手动轨迹点精修、轨迹分组、环境设置、群众、颜色、飞行速度、全屏画布

## 3. 层级规则

- **L1 常驻**（不看选中、全程可见）：出片（目标锚点）、添加（建造锚点）、帮助、关闭
- **L2 情境**（随选中/阶段自动出现，出现时机=需要时机）：右栏属性+整运镜（选中相机）、姿势（选中假人）、相机预览卡（选中相机）、时间轴（第一段运镜诞生）、接控入口（选中可操控对象）
- **L3 模式内**（进入模式才出现，带退出 banner）：轨迹编辑工具、接控条(动作库+REC+速度)、取景调整
- **L4 收纳**（低频进菜单/悬浮，不占常驻位）：变换模式(视口悬浮 pill+快捷键)、环境设置、全屏
- 铁律：**一个功能一个家**（重复入口删除）；**隐藏手势必须有可见说明**（双击加点写进轨迹 tab 空态）

## 4. 布局映射（区域=语义）

```
顶栏 = 身份与出口     ：3D场景标题 ｜ (模式 banner 位) ｜ 出片 · ? · ×
左栏 = 场景里有什么   ：场景节点树（不变）
视口 = 手上正在弄的   ：3D + 悬浮：变换 pill(左上) · 相机预览卡(右上) · 模式 banner(顶中)
右栏 = 选中对象的一切 ：属性(上) + 整运镜三 tab(下，选中相机时)
底部 = 建造与时间     ：添加 · 全屏(收纳) ｜ 时间轴(时机出现) ｜ 接控条(模式内)
```

顶栏 10 控件 → 4：
- 移动/旋转 → 视口左上悬浮 pill（贴操作对象，DCC 惯例）+ W/E 快捷键
- 截图 → 删（出片面板已有视口/相机截图，重复入口）
- 轨迹 toggle → 整运镜>轨迹 tab 的「进入编辑」（同源收拢）
- 播放 → 时间轴（它出现的时机=可播放的时机；顶栏播放删）
- 接控 → 随选中出现（对象头顶/右栏），录 take 走整运镜>录take tab
- 速度滑杆 → 接控条内（只在飞行时有意义）
底部「轨迹」钮 → 删（同源收拢进整运镜）。
相机预览卡的画幅/焦段/镜深/抖动**留在卡上**（取景器心智：调参立刻看到画面变化，搬去右栏伤体验——此为有意取舍非遗漏）。

一级可见控件 37 → 约 22；顶栏认知负担降 60%。

## 5. 动线验证（新布局跳数）

| 任务 | 旧 | 新 |
|---|---|---|
| 想动镜头（不知道从哪动） | 3 处入口自己找 | 右栏整运镜区固定在那 |
| 预设落轨→出片 | 选相机→滚到底→预设→找出片 | 选相机→预设(第一屏)→接力亮→出片 |
| 录 take | 选假人→接控(顶栏)→底部 REC | 整运镜>录take → 开始录制 |
| 切移动/旋转 | 顶栏来回够 | 视口左上就近点 / W·E |
| 播放预览 | 顶栏或时间轴（两处） | 时间轴一处（出现即可播） |

## 6. 分期实施（每期独立 gates+走查，可独立回滚）

- **一期（拍板即做）**：整运镜三合一（右栏轨迹 tab 删除内容搬入 + 录take tab）＋顶栏删重复（截图/播放/轨迹 toggle）
- **二期**：变换 pill 移视口悬浮＋速度进接控条＋接控随选中＋底部轨迹钮删
- **三期**：走查全旅程数跳数复核 + 文案统一 + 重看引导覆盖新布局（coach 锚点更新）

## 7. 不做

- 不改视口手势（WASD/滚轮/双击语义不动，只补说明）
- 不动左栏结构、不动出片面板、不动时间轴出现时机（刚按旅程修完）
- 不做自定义快捷键/布局记忆（等真实反馈）
