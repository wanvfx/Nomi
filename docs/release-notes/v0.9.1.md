# v0.9.1 — 修音频导出（打包 ffprobe）

发布日期：2026-06-04

## 一句话

**让 v0.9.0 的音频导出对所有用户真正生效。**

## 修复

- **打包 ffprobe**：v0.9.0 的音频导出依赖 ffprobe 探测视频是否带音轨，但安装包只随附了 ffmpeg、没带 ffprobe → 分发版用户探测失败、音频静默回退静音（开发机因系统装了 ffprobe 而未暴露）。本版随附 `@ffprobe-installer/ffprobe` 并优先使用，含音轨的视频导出后 MP4 真正带声。
- 新增真实 ffmpeg 端到端测试脚本（合成带音轨视频 → filtergraph 导出 → 校验产物含音视频流）。

> 升级建议：v0.9.0 用户请更新到 v0.9.1 以获得可用的音频导出。
