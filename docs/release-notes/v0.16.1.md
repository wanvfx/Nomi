# Nomi v0.16.1

Patch release focused on making the 3D director reference workflow more reliable.

## Highlights

- Recalibrated mannequin pose presets for sitting, squatting, crouching, kneeling, pointing, waving, and cheering.
- Fixed Pose Lab full-shot coverage so all pose presets are captured automatically instead of relying on hard-coded batches.
- Added regression coverage to prevent new pose presets from being skipped by the screenshot tool.
- Updated the public README and website community QR code.

## Validation

- Pose Lab screenshots regenerated across front / side / q3 / back / top views.
- 31 staging reference scenes rendered across 6 views each.
- Typecheck, lint, unit tests, and production build passed.
