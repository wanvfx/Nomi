第一次使用 Nomi

这不是示例项目选择器，只带你走一遍真实工作流。完成后首页不再显示大块引导。

1. 接入文本模型让创作助手和 Agent 可用。
2. 创建项目确认脚本、素材、导出保存位置。
3. 走完工作区创作 → 生成 → 预览导出。

C 把 Onboarding 完全移进弹窗/流程，只给首页留下一个“快速了解流程”入口。日常最干净，但首次用户可能更依赖弹窗出现时机。
