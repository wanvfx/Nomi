C：引导发生在真实工作台，锚定真实内容（示例故事），不是空白页讲概念。不做 scrim 压暗（避免「让用户看的内容反而变暗 / 被挡」+ 挖洞工程），只用焦点区 accent 描边 + spotlight 真实按钮；callout 锚在画布空白处，几何上不得遮挡故事文字与焦点按钮（落 design-fidelity 断言）。推进 = 点真实的「拆成分镜」，它是一次真实 LLM 调用，必须定义两个状态：调用中（callout 变「正在拆分镜…」）、失败（callout 给「重试 / 跳过本步」）。跳过 / 完成写入持久化标记（新引入，现仓没有）；重看入口 = appbar 右侧「?」（开放项，也可并入设置）。从 B 点「新建空白项目」进来的缺模型用户：创作助手面板内显示与 A2 同款状态条。
